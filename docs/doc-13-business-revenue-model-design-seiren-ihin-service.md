# 清蓮｜遺品整理サービス 事業モデル・収益設計書

**文書名**：清蓮｜遺品整理サービス 事業モデル・収益設計書  
**対象サービス**：清蓮｜遺品整理サービス

---

## 1. 文書概要

**目的**  
本書は清蓮の遺品整理サービスにおける事業構造、収益モデル、役割分担、顧客導線、収益最大化戦略を定義する。

本サービスは **清蓮** と **株式会社ベルコール** の共同運営型モデルとする。

- 清蓮：集客 / 相談窓口 / 顧客対応
- ベルコール：見積 / 契約 / 作業

清蓮はバックマージンによる収益を得る。

---

## 2. 事業スキーム

```
顧客 → 清蓮サイト → 清蓮ヒアリング → ベルコール見積 → ベルコール契約 → 作業 → 清蓮へ紹介料
```
