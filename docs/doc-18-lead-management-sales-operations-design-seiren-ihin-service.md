# 清蓮｜遺品整理サービス リード管理・営業オペレーション設計書

**文書名**：清蓮｜遺品整理サービス リード管理・営業オペレーション設計書  
**対象サービス**：清蓮｜遺品整理サービス

---

## 1. 文書概要

**目的**  
本書は、清蓮遺品整理サービスにおける問い合わせ（リード）獲得後の営業フロー、顧客管理、案件進行管理、ベルコールとの連携ルールを定義する。

遺品整理ビジネスでは問い合わせ → 見積 → 契約 → 作業の管理が曖昧になると、機会損失・顧客不信・成約率低下・紹介料漏れが発生する。

**本設計書の定義範囲**

- リード管理 / 顧客対応 / 営業フロー
- 案件ステータス / ベルコール連携 / 成約管理

---

## 2. 基本営業構造

```
顧客 → 清蓮サイト問い合わせ → 清蓮ヒアリング → ベルコール見積 → 顧客契約 → 作業 → 清蓮紹介料
```

---

## 3. リード（問い合わせ）の定義

電話・フォーム・LINE・紹介などから取得した遺品整理検討顧客を指す。

---

## 4. リード取得チャネル

サイト問い合わせフォーム / 電話 / LINE相談 / SEO流入 / 広告 / 紹介 / 葬儀社 / 終活相談

---

## 5. リード情報項目

| 区分     | 項目                                       |
| -------- | ------------------------------------------ |
| 必須情報 | 名前 / 電話番号 / メール / 住所 / 作業場所 |
| 案件情報 | 間取り / 荷物量 / 希望日程 / 特殊清掃有無  |
| 補足情報 | ゴミ屋敷 / 遺品整理 / 供養相談             |

---

## 6. リード管理ステータス

1. 新規問い合わせ
2. ヒアリング中
3. 見積依頼済
4. 見積提出
5. 検討中
6. 成約
7. 失注
8. 作業完了
9. 紹介料確定

---

## 7. リード管理フロー

```
問い合わせ受信 → 清蓮ヒアリング → ベルコール見積依頼 → 現地見積 → 見積提出 → 契約 → 作業 → 紹介料計算
```

---

## 8. 清蓮ヒアリング項目

### 8-1. 基本情報

名前 / 電話 / 住所

### 8-2. 作業情報

間取り / 荷物量 / 大型家具 / エレベーター

### 8-3. 特殊条件

ゴミ屋敷 / 特殊清掃 / 孤独死

### 8-4. スケジュール

希望日 / 急ぎか

---

## 9. ヒアリングスクリプト

最初の対応は **丁寧・安心・誠実** を基本とする。

**確認内容**：作業場所 / 間取り / 荷物量 / 作業希望日 / 現地見積希望

---

## 10. ベルコール連携

**清蓮からベルコールへ共有する情報**  
顧客名 / 住所 / 連絡先 / 間取り / 荷物量 / 希望日

**共有方法**：電話 / LINE / メール

---

## 11. 見積フロー

- **現地見積**：ベルコール担当
- **見積書**：ベルコール名義
- **清蓮**：顧客フォロー

---

## 12. 顧客フォロー

- 見積提出後 **2日以内** にフォロー
- **フォロー内容**：質問確認 / 検討状況確認

---

## 13. 成約率向上施策

安心説明 / 料金透明化 / 施工事例提示 / 供養案内

---

## 14. 失注理由管理

高い / 他社契約 / 日程不一致 / 家族相談

---

## 15. 失注分析

毎月分析

---

## 16. 成約管理

- **成約条件**：顧客とベルコール間で契約締結

---

## 17. 作業管理

作業日の確認

---

## 18. 紹介料管理

紹介料 = 作業金額 × 紹介率

---

## 19. 売上管理

案件を月別管理

---

問い合わせ数 / 見積数 / 成約数 / 成約率

30%
