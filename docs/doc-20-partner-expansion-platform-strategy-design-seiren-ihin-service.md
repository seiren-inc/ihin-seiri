# 清蓮｜遺品整理サービス パートナー拡張・プラットフォーム戦略設計書

**文書名**：清蓮｜遺品整理サービス パートナー拡張・プラットフォーム戦略設計書  
**対象サービス**：清蓮｜遺品整理サービス

---

## 1. 文書概要

**目的**  
本書は清蓮遺品整理サービスを、単一業者紹介モデルから将来的に複数業者が参加する終活サービスプラットフォームへ拡張するための設計を定義する。

**現在の体制**

| 主体               | 役割               |
| ------------------ | ------------------ |
| 清蓮               | 集客 / 相談窓口    |
| 株式会社ベルコール | 見積 / 契約 / 作業 |

**将来の体制**

| 主体     | 役割                 |
| -------- | -------------------- |
| 清蓮     | プラットフォーム運営 |
| 地域業者 | 作業実施             |

紹介ビジネスから **終活プラットフォーム** への進化を想定する。

---

## 2. 現在の事業構造

```
顧客 → 清蓮サイト → 清蓮ヒアリング → ベルコール見積 → 契約 → 作業 → 清蓮紹介料
```

## 3. 拡張後モデル

