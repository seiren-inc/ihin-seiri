## プロジェクト概要

### サイト名

清蓮｜遺品整理サービス

### 事業スキーム

**清蓮が担当**：集客 / 相談窓口 / 初期ヒアリング / ブランド統括 / 供養導線設計

**株式会社ベルコールが担当**：見積 / 契約 / 実作業

サイト上では以下を徹底すること：

- 清蓮は相談窓口である
- 見積書は提携専門業者名義で発行される

- 供養相談連携（仏壇、位牌、遺骨、粉骨、散骨など清蓮既存サービスへの導線）

## サイトの本質

このサイトは「安売りの片付け業者サイト」ではなく **「終活相談プラットフォームの入口」** として構築すること。

### 競合との差別化軸

- 供養導線がある
- 料金シミュレーションがある
- 事例が見やすい
- 関東広域対応
- モダンで洗練されたUI
- 高齢者にも見やすい
- 信頼・透明性・安心感が強い
- SEOとGEOに強い
- 将来的な地域ページ量産、業者拡張、管理画面運用に耐える

---

## 絶対条件

1. 古い遺品整理業者サイトのようなデザインにしないこと
2. チラシ風、激安訴求、過剰バナー、安っぽい色使いは禁止
3. 文字を詰め込みすぎず、余白を活かしたモダンなデザインにすること
4. Appleのような整理感 / 終活サービスとしての落ち着き / 行政サイトのような信頼性を両立したUIにすること
5. スマートフォン最優先で設計すること
6. 全ページに画像を入れる前提で、指定箇所に空ホルダーを配置すること（画像が未確定でも画面構造が成立するようにすること）
7. ページ表示速度、SEO、アクセシビリティを犠牲にする重い実装は禁止

## ページごとの実装指針

- **トップ**：最も信頼感を形成するページ。過度な説明より、見出しとカードで理解しやすくする。料金・事例・供養・対応エリアへ自然に誘導する。
- **サービスページ**：遺品整理・ゴミ清掃・特殊清掃それぞれの違いが明確に分かる構造にする。
- **料金ページ**：料金表だけでなく、何で金額が変わるか / 何が含まれるか / 追加費用が起きやすい条件を明示する。
- **事例一覧**：絞り込みを見やすく、一覧性を高く、事例詳細へ自然に導線を作る。
- **事例詳細**：ビフォーアフター / 料金帯 / 間取り / 作業時間 / ポイント / 供養相談有無を入れて信頼形成に使う。
- **地域ページ**：地名を自然に含む。地域向け説明 / 対応サービス / 地域事例 / FAQ / CTAをテンプレート化して量産しやすくする。
- **FAQ**：カテゴリ別アコーディオン形式。検索意図を意識した質問文にする。
- **お問い合わせ**：フォーム / 電話 / LINEを明確に分ける。送信ハードルを下げる。送信後の安心感も設計する。

## 品質基準

- 高級感ではなく上質感
- 派手さではなく信頼感
- わかりやすさを最優先
- SPで使いにくいUIは禁止
