# 清蓮｜遺品整理サービス Webサイト AI機能設計書

**文書名**：清蓮｜遺品整理サービス AI機能設計書  
**対象サイト**：清蓮｜遺品整理サービス

---

## 1. 文書概要

**目的**  
本書は清蓮の遺品整理サービスサイトにおいて、競合サイトには存在しない先進的なAI機能を導入し、ユーザー体験・問い合わせ率・信頼性を向上させることを目的とする。

遺品整理業界のサイトは料金表・事例・問い合わせ程度の機能しかない。  
本サイトでは **AI・自動診断・シミュレーション** を導入し、ユーザーの不安を解消しながら問い合わせへ誘導する。

---

## 2. AI導入方針

本サイトのAI機能は以下を目的とする。

- 料金の不透明さを解消
- 相談のハードルを下げる
- 問い合わせ率向上
- ユーザー理解向上
- SEO強化 / GEO強化
