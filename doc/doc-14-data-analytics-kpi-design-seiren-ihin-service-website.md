## 1. 文書概要

**目的**  
本書は清蓮の遺品整理サービスサイトの運用において、データ分析、KPI設定、改善サイクルを定義する。

本サイトは公開後の **アクセス分析・問い合わせ分析・SEO分析・ユーザー行動分析** を行うことで問い合わせ率を最大化する。

**本書の定義範囲**

- KPI設計
- 分析指標
- 分析ツール
- 改善サイクル
- レポート運用

---

## 2. 分析の目的

## 3. KPI構造

本サイトのKPIは3階層で設計する。

| 階層      | 内容                                |
| --------- | ----------------------------------- |
| 事業KPI   | 問い合わせ・見積・成約              |
| サイトKPI | 検索流入・訪問数・CV数              |
| ページKPI | クリック率・滞在時間・CTAクリック率 |

---

## 4. 事業KPI

- **最終KPI**：問い合わせ数
- **補助KPI**：見積数 / 成約数

## 5. サイトKPI
