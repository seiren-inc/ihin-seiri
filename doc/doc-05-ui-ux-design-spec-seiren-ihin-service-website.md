## 1. 文書概要

**目的**  
本書は、要件定義書、基本設計書、詳細設計書、技術設計書を基に、サイトのUI設計、UX設計、ビジュアルデザインルール、コンポーネントデザイン、ユーザー体験フローを定義することを目的とする。

本サイトは遺品整理というセンシティブなサービスを扱うため、**信頼性、安心感、理解のしやすさ、相談しやすさ**を最優先とする。

実装はAntigravityで行う。  
本設計書はUI・UX実装の基準書として扱う。

## 2. UX設計の基本思想

### 2-1. UXコンセプト

**ユーザーの不安を先回りして解消するサイト**

遺品整理のユーザーは以下の心理状態にある。

- 何から始めていいかわからない
- 料金が不安
- 業者に騙されないか不安
- 遺品の扱いが雑ではないか不安
- 家族の思い出をどう扱うか迷っている
- 急いでいるが慎重にもなりたい

本サイトでは以下のUX設計を徹底する。

- 安心感
- 透明性
- 理解しやすさ
- 相談のしやすさ
- 供養まで含めた信頼性

### 2-2. UX目標

ユーザーが最短3分で以下を理解できる状態を作る。

1. 何を依頼できるか
2. いくらくらいかかるか
3. どの会社が何をするのか
4. どの地域で対応できるのか
5. どう相談すればいいのか

その上で問い合わせ行動へ誘導する。

### 2-3. UX指標

- 問い合わせ率
- 料金シミュレーション利用率
- 施工事例閲覧率
- 電話クリック率
- LINE相談率
- FAQ閲覧率

## 24. 最終UX方針

本サイトは遺品整理業者サイトではなく、**終活相談サイト**として設計する。  
ユーザーの不安を軽減し、信頼を形成し、自然に相談へ導くUX設計を徹底する。
