# 清蓮｜遺品整理サービス インシデント対応・危機管理設計書

**文書名**：清蓮｜遺品整理サービス インシデント対応・危機管理設計書  
**対象サービス**：清蓮｜遺品整理サービス

---

## 1. 文書概要

**目的**  
本書は、清蓮｜遺品整理サービスにおいて発生し得る事故、苦情、炎上、法務問題、現場トラブル、情報漏えい、業務停止リスクに対し、初動、報告、責任分界、顧客対応、再発防止までの対応基準を定義することを目的とする。

本サービスは共同運営型であり、清蓮が集客・相談窓口・初期ヒアリングを担い、株式会社ベルコールが見積・契約・実作業を担う。

そのため、トラブル時に **誰が何をするのか・誰が顧客に説明するのか・誰が責任主体か** を明確にしておかないと、顧客不信、ブランド毀損、紹介料紛争、法務リスクが発生する。

---

## 2. 基本方針

### 2-1. 基本原則

- 初動を最優先する
- 事実確認前に断定しない
- 責任主体と説明主体を分けて整理する
- 顧客を放置しない
- 口頭だけで終わらせず記録を残す
- 清蓮ブランド保護と顧客保護を両立する
- 再発防止まで完了として扱う

### 2-2. 対応優先順位

1. 人命・安全
2. 個人情報・法務
3. 現場事故・損害
4. 顧客感情・信用失墜
5. 収益影響
6. 再発防止

### 2-3. 原則姿勢

- 隠さない / 逃げない / 誇張しない
- 感情的に反論しない
- 証拠を残す
- 社内判断と顧客説明を分ける

| Level   | 区分     | 概要                                               | 例                                                                         |

### 4-4. 法務・許認可系

### 4-5. 情報管理系

## 5. 役割分担

### 5-1. 清蓮の役割

- 一次窓口対応 / 初動連絡
- 顧客感情の一次受け止め
- 事実整理
- ベルコールへの即時連携
- サイト表記・導線に関する問題修正
- ブランド毀損リスク管理
- 必要に応じて謝罪窓口

### 5-2. ベルコールの役割

- 見積・契約・作業に関する一次責任主体
- 現場事実確認 / 作業員ヒアリング / 証拠収集
- 損害対応 / 返金・再作業・補償協議
- 契約条項に基づく説明

### 5-3. 共同対応事項

- 重大クレーム対応方針の統一
- 顧客への最終説明調整
- 再発防止策の策定
- 必要時の文書回答

### 5-4. 顧客への説明原則

- 清蓮は窓口として連絡を受ける
- 作業・契約に関する正式説明はベルコール主体
- ただし顧客から見ると一体サービスに見えるため、たらい回しにしない

## 6. 初動対応フロー

### 6-1. 基本フロー

```
インシデント発生 → 受信者が内容記録 → 緊急度判定 → 関係者通知
→ 顧客へ一次返信 → 事実確認 → 暫定対応 → 正式回答
→ 再発防止策策定 → クローズ記録
```

### 6-2. 初動時間基準

| Level   | 対応基準                                                                           |
| ------- | ---------------------------------------------------------------------------------- |
| Level 1 | 当日中                                                                             |
| Level 2 | 2時間以内に一次返信、当日中に事実確認開始                                          |
| Level 3 | 1時間以内に社内共有、2時間以内に顧客へ一次返信、当日中に責任者判断                 |
| Level 4 | 即時共有、30分以内に関係者招集、顧客への一次連絡を最優先、必要に応じて公開対応準備 |

### 6-3. 一次返信の原則

- 事実確認中であることを伝える
- 不安に対する共感を示す
- 放置しない
- 責任を確定前に断定しない
- 再連絡時刻の目安を伝える

## 7. エスカレーション設計

### 7-1. エスカレーション条件

- 損害が発生している
- 顧客が強い不信感を示している
- 返金要求がある
- SNSや口コミへの投稿が示唆されている
- 法務リスクがある
- 個人情報が含まれる
- 現場判断で収束不能

### 7-2. エスカレーション先

清蓮責任者 → ベルコール責任者 → 顧問弁護士 → 保険窓口 → 行政・警察等

### 7-3. 緊急連絡ルール

- 電話優先、その後に文書化
- 重要事項はLINEだけで完結しない
- 最終判断はテキストで残す

---

## 8. 顧客対応ルール

### 8-1. 基本ルール

- 反論から入らない
- 顧客の主張を一旦受け止める
- 記録を取りながら聞く
- 曖昧な約束をしない
- その場しのぎの値引きを乱発しない
- 責任主体不明のまま押し付け合わない

### 8-2. 禁止事項

- 感情的な言い返し / 言い訳優先
- ベルコールのせいにするだけの説明
- 「清蓮は関係ない」という切り方
- 事実未確認の断定
- 無断既読スルー
- 謝るべき場面で謝らない

### 8-3. 一次対応テンプレ要素

- ご不快な思いへの謝意
- 状況確認中である旨
- 担当確認の実施
- 再連絡予定
- 緊急時連絡先

## 9. インシデント別対応設計

### 9-1. 見積・金額トラブル

**想定**：見積より高い / 追加費用の説明がなかった / シミュレーションと違う

**対応**：シミュレーション画面と注記確認 → 現地見積内容の確認 → 追加理由の根拠確認 → ベルコール見積書記載確認 → 顧客への比較説明 → 必要に応じて調整案提示

**再発防止**：サイト表現修正 / 見積前説明文の統一 / 追加費用項目の事前案内強化

### 9-2. 契約主体誤認トラブル

**想定**：清蓮と契約したと思っていた / ベルコールの存在を知らなかった

**対応**：サイト表記確認 → フォーム後自動返信文面確認 → 見積提出時の説明有無確認 → 顧客へ整理して説明 → 必要に応じて謝意表明

**再発防止**：トップ・料金・FAQ・問い合わせでの主体表記強化 / 見積案内文の定型化

### 9-3. 遺品・貴重品紛失

**対応**：最優先で重大扱い → 作業記録・写真・担当者証言確認 → 持ち出し導線確認 → 顧客申告内容を記録 → ベルコール主体で正式調査 → 保険適用確認 → 必要なら弁護士相談

**再発防止**：作業前確認書 / 探索物チェックルール / 写真記録ルール強化

### 9-4. 破損事故

**想定**：家具・壁・床・共用部・エレベーター破損

**対応**：現場写真取得 → 日時・状況・破損箇所記録 → 建物管理者対応有無確認 → ベルコール保険確認 → 顧客へ調査中説明 → 補修・賠償案整理

**再発防止**：養生ルール / 搬出ルール / 作業前チェックリスト整備

### 9-5. 特殊清掃品質クレーム

**想定**：臭いが残る / 清掃不足 / 期待値相違

**対応**：作業範囲と契約内容確認 → 現地再確認 → 再施工可能範囲整理 → 専門説明の補足 → 現場状態と限界値を説明

**再発防止**：契約前に仕上がり期待値を明文化 / 説明文改善

### 9-6. 個人情報漏えい

**想定**：見積書誤送信 / 住所・電話番号の共有ミス / 写真送付先ミス

**対応**：即時Level 4扱い候補 → 送信停止 → 回収依頼 → 対象範囲特定 → 関係者報告 → 顧客へ速やかに説明 → 二次被害防止措置

**再発防止**：送信前ダブルチェック / 個人情報マスキング / 権限管理見直し

### 9-7. 口コミ・SNS炎上

**対応**：事実確認前に公開反論しない → 投稿内容保存 → 顧客特定 → 対応履歴確認 → 必要なら個別連絡 → 公開返信は短く誠実に → 個人情報に触れない

**再発防止**：返信テンプレ整備 / 現場教育 / 顧客説明改善

## 10. 記録管理

### 10-1. 必須記録項目

発生日 / 受付日時 / 受付者 / 顧客名 / 案件番号 / 発生場所 / 概要 / 証拠 / 一次対応内容 / ベルコール共有日時 / 顧客再連絡日時 / 最終対応 / 再発防止策 / クローズ日

### 10-2. 証拠保全

スクリーンショット / 写真 / 見積書 / LINE履歴 / メール / 通話メモ / 作業記録

### 10-3. 保存ルール

- 案件単位で一元化
- 口頭報告のみ禁止
- 修正履歴を残す

## 11. 公開・対外対応

### 11-1. 外部発信基準

| Level      | 対応                                   |
| ---------- | -------------------------------------- |
| Level 1〜2 | 原則非公開対応                         |
| Level 3    | 必要に応じて個別文書対応               |
| Level 4    | 公開説明文、弁護士相談、プレス対応検討 |

### 11-2. 口コミ返信ルール

- 感情的にならない / 個別事情に触れない / 確認中である旨 / 窓口誘導 / 謝意を示す

### 11-3. SNS投稿ルール

- 現場トラブル中は勝手に投稿しない
- 対応完了前の自社アピール投稿禁止
- 顧客写真や現場写真の扱いに注意

## 18. 最終方針

本サービスでは、トラブルをゼロにすることではなく、**発生時に迅速かつ誠実に収束させ、顧客不信とブランド毀損を最小化する**ことを危機管理の基本方針とする。

清蓮は窓口として逃げずに受け止める。  
ベルコールは契約・作業主体として責任ある調査と是正を行う。  
両者が役割を明確にしながら、一体として顧客に向き合う体制を標準運用とする。
